## 2. 온(화폐) 아이콘·표기 일관성

가상 화폐를 '콩'에서 '온'으로 변경했으나, 아래에서 여전히 콩 아이콘(🫘)이 사용됨.

| 위치 | 현재 | 수정 제안 |
|------|------|-----------|
| `BannerWriteModal.jsx` L41 | 🫘 {cost} | ⚡ {cost} 온 |
| `ActivityRewardCenter.jsx` L44 | 🫘 (리워드 애니메이션) | ⚡ |
| `GangnamRomance.jsx` L229 | -${cost} 🫘 | -${cost} 온 또는 ⚡ |
| `RightPanel.jsx` L383 | 🫘 | ⚡ |
| `AvatarCustomizer.jsx` L99, L231 | 🫘 | ⚡ |

## 4. 사용자 경험(UX) 개선 제안

| 항목 | 현재 | 제안 | 우선순위 |
|------|------|------|----------|
| 배너 글자 수 | maxLength 50자 | 핫플/알림성 문구에 짧을 수 있음. 80~100자 검토 | 낮음 |
| 플로우 배너 등록 | placeholder만 안내 | 예시 문구 2~3개 제시 (선택 가능) | 낮음 |
| 라운지 유료 콘텐츠 | 1온 소모, 확인창 | 버튼에 "(1온)" 표시로 사전 인지 용이 | 중간 |
| 강남 로맨스 심쿵 | -5온만 표시 | 슈퍼라이크 등 다른 액션 비용도 명시 | 낮음 |
| 에러 바운더리 | 기술적 에러 노출 | 사용자 친화적 메시지 + "다시 시도" 유도 | 중간 |

## 7. 요약: 즉시 수정 권장 항목

1. **Safegauard** → Safeguard (`MiniHomepage.jsx`)
2. **저녁 해돋이** → 저녁 노을 / 일몰 (`App.jsx`)
3. **콩 아이콘(🫘)** → 온 아이콘(⚡) (`BannerWriteModal`, `ActivityRewardCenter`, `GangnamRomance`, `RightPanel`, `AvatarCustomizer`)
4. **BannerWriteModal** 비용 표기: `🫘 {cost}` → `⚡ {cost} 온`

이 항목들을 반영하면 오타·일관성·UX가 개선됩니다.
