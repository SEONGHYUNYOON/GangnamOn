## 무엇이 바뀌었나
1. **카드 클릭 시 상세 모달 추가** — 지금까지는 카드를 눌러도 아무 반응이
   없었는데, 클릭하면 사진 전체(여러 장이면 갤러리로 넘겨보기), 전체 소개
   문구(잘림 없이), 주소(네이버 지도로 바로 연결), 전화번호, "원문 블로그에서
   자세히 보기" 버튼이 있는 모달이 뜨도록 함 (src/components/GangnamPickBoard.jsx)
2. **좋아요 버튼 실제로 클릭되게 수정** — 예전엔 하트 아이콘이 그냥 숫자
   표시용이라 눌러도 아무 반응이 없었음. 이제 누르면 채워지고 카운트가
   바뀌는 인터랙션이 생김 (참고: 이 앱의 다른 좋아요 버튼들도 전부 DB에
   저장되지 않는 로컬 표시용이라, 이번에도 같은 방식으로 통일함 — 실제
   서버 저장이 필요하면 별도로 알려줘)
3. **사진 수집 안정성 개선** (functions/blogFeed/src/main.js)
   - 사진을 찾는 이미지 도메인 필터를 pstatic.net 외에 naver.net까지
     넓힘 (일부 블로그 사진이 다른 네이버 CDN 도메인을 써서 못 찾던 문제)
   - 블로그 링크 형식을 경로형(blog.naver.com/아이디/글번호) 외에
     쿼리파라미터형(?blogId=...&logNo=...)도 지원하도록 파싱 로직 보강
   - 사진을 하나도 못 찾았을 때의 대체 수단을 PC 페이지 og:image 1개에서
     "PC 페이지 시도 → 실패하면 모바일 페이지도 시도"로 이중화
   - 최대 수집 장수를 3장 → 5장으로 늘림
