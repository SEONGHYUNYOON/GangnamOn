# 카카오 지도 API 도메인 등록 가이드

강남온(GangnamOn) 서비스에서 지도가 정상적으로 보이지 않는다면, 배포된 주소(**vercel.app**)를 카카오 개발자 센터에 등록해주어야 합니다.
아래 순서대로 따라하시면 1분 안에 해결됩니다!

## 1. 카카오 개발자 센터 접속

1. [카카오 개발자 센터(https://developers.kakao.com/)](https://developers.kakao.com/) 링크를 클릭하여 접속합니다.
2. 우측 상단의 **[로그인]** 버튼을 눌러 카카오 계정으로 로그인합니다.
3. 로그인 후, 상단 메뉴의 **[내 애플리케이션]**을 클릭합니다.

## 2. 애플리케이션 선택

1. 목록에 있는 **[GangnamOn]** (또는 지도 키를 발급받았던 앱 이름)을 클릭합니다.
   - *팁: 만약 앱이 없다면, '애플리케이션 추가하기'를 눌러 새로 만드셔도 됩니다.*

## 3. 플랫폼 설정 이동

1. 왼쪽 메뉴 리스트에서 **[플랫폼]**을 클릭합니다.
2. 화면 중앙에 **Web** 항목이 보일 것입니다.
   - 만약 'Web 플랫폼을 등록해 주세요'라고 되어 있다면 **[Web 플랫폼 등록]** 버튼을 누릅니다.
   - 이미 등록되어 있다면 **[수정]** 버튼을 누릅니다.

## 4. 사이트 도메인 등록 (가장 중요! ⭐)

**사이트 도메인** 입력칸에 아래 주소들을 모두 입력해주세요. (줄바꿈으로 구분)

```text
http://localhost:5173
https://gangnam-on.vercel.app
```

*(위 주소들을 복사해서 붙여넣으시면 됩니다)*

- `http://localhost:5173`: 로컬 개발 환경에서 지도를 보기 위함
- `https://gangnam-on.vercel.app`: **실제 배포된 사이트**에서 지도를 보기 위함

입력이 끝났다면 하단의 **[저장]** 버튼을 클릭합니다.

## 5. 확인하기

1. 설정이 완료되었습니다!
2. 이제 [강남온 사이트](https://gangnam-on.vercel.app)에 다시 접속해서 **새로고침(F5)**을 해보세요.
3. 지도가 정상적으로 나오는지 확인합니다.
