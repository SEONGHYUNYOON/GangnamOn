# 강남온 네이티브 앱 (GangnamOn Native)

웹 서비스 GangNam_On 기반 iOS / Android 네이티브 앱 프로젝트입니다.

## 구조

- **ios/** — Swift + SwiftUI (iOS)
- **android/** — Kotlin + Jetpack Compose (Android)

## 실행 방법

### iOS
1. Xcode에서 `ios/GangnamOn.xcodeproj` 열기
2. 시뮬레이터 또는 연결된 기기 선택 후 Run (⌘R)

### Android
1. Android Studio에서 `android/` 폴더 열기
2. Gradle 동기화 후 에뮬레이터 또는 연결된 기기 선택 후 Run
3. 하단 탭: **홈**(모임 피드), **동네생활**, **중고거래**, **미니홈피**, **마이**

## 요구사항

- **iOS**: Xcode 15+, macOS, iOS 15+
- **Android**: Android Studio Hedgehog+, JDK 17, Android SDK 34
